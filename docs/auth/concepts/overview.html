Overview

This section explains the core concepts behind authentication and authorization in ERP.net: identity authorities, OAuth 2.0, and how permissions are expressed and enforced.

If you're looking for step-by-step scenario guides (SPA, web apps, service integrations), start with Getting Started. If you need instance-side setup, see Configuration.
