The Domain API is based on the OData protocol. It allows object-oriented access to the data exposed by the ERP.net domain objects.

Since the Domain API is an HTTP RESTfull API, it can be accessed by simple HTTP requests. However there are many ODATA libraries that can be used.

ERP.net libraries

.NET

ErpNet.Api.Client - Handles authentication of service clients and work with Domain API requests, both - untyped and typed.

Javascript

oidc-client - Library to provide OpenID Connect (OIDC) and OAuth2 protocol support for client-side, browser-based JavaScript client applications. Also included is support for user session and access token management.

PHP

OpenIdConnect client - A simple library that allows an application to authenticate a user through the basic OpenID Connect flow.
